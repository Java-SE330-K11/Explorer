## Thông tin về dự án

Phần mềm Explorer nằm trong đồ án cuối kỳ môn Ngôn ngữ lập trình Java SE330.K11.

### Nền tảng

Phần mềm được viết cho nền tảng Windows bằng ngôn ngữ Java, sử dụng IDE Netbeans.

Phần mềm này sử dụng Java Swing.
